# NEXUS AI DIGITAL – Frontend Internship Task

## Project Overview

A **responsive, interactive landing page** built as part of the **NEXUS AI DIGITAL Frontend Web Development Internship (Part I: The UI Specialist)**.

This single-page application demonstrates **clean UI design**, **modern front-end fundamentals**, and **real-world deployment** using **HTML5, CSS3, and Vanilla JavaScript** — **no frameworks**.

---

## Features

| Feature                  | Implemented |
|--------------------------|-------------|
| Semantic HTML5           | Yes         |
| Responsive Design        | Yes (Mobile, Tablet, Desktop) |
| CSS Flexbox Layout       | Yes         |
| Smooth Scroll            | Yes         |
| Hover Effects            | Yes         |
| Typewriter Animation     | Yes (Vanilla JS) |
| Dark Mode Toggle         | Yes (CSS Variables + JS) |
| Fast Loading (CDN-free)  | Yes         |
| Deployed on Netlify      | Yes         |

---

## Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Flexbox, Grid, Transitions, Variables
- **Vanilla JavaScript** – Typewriter, Dark Mode, Smooth Scroll
- **Google Fonts** – Poppins
- **Netlify** – Hosting & HTTPS
- **GitHub Pages** – Backup deployment
